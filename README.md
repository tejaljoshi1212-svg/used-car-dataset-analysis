# used-car-dataset-analysis
A data analysis project on used car dataset to understand pricing trends, mileage, and other factors affecting car prices.”
Project Overview

This project is based on a used car dataset and aims to analyze different factors affecting car prices such as mileage, brand, fuel type, and year of manufacture. The goal is to gain meaningful insights from the data and understand pricing trends.

Objective:
1-Used car prices ko analyze karna
2-Important factors identify karna jo price affect karte hain
3-Data visualization ke through insights nikalna

- Tools & Technologies Used
Python 
Pandas
NumPy
Matplotlib / Seaborn
Jupyter Notebook

Dataset Features
Car Brand
Price
Year
Mileage
Fuel Type
Transmission

Key Insights
Older cars generally have lower prices
Mileage price ko directly affect karta hai
Certain brands retain value better than others
